Resources

The GitHub repository contains the released CoFlow source code. The Hugging Face dataset mirror provides a convenient download entry, and the original public dataset sources are listed below.

CoFlow GitHub Repository Official source-code release with training, evaluation, experiment configs, and setup instructions. Open code repository
CoFlow Dataset Mirror Hugging Face dataset entry for convenient access to the offline MARL datasets used by the released code. Open Hugging Face dataset
MPE Datasets Expert, Medium, Medium-Replay, and Random splits from the OMAR offline MARL benchmark. Open OMAR source
MA-MuJoCo and SMAC Datasets Public MADiff mirror of off-the-grid MARL datasets used for locomotion and StarCraft experiments. Open MADiff dataset mirror
OG-MARL Benchmark Original off-the-grid MARL benchmark behind the MA-MuJoCo and SMAC offline splits. Open OG-MARL benchmark

Related Work

  • MADiff: diffusion-based offline multi-agent trajectory generation.
  • OG-MARL: off-the-grid offline multi-agent benchmark suites.
  • OMAR: offline multi-agent actor-critic learning with first widely used MPE offline splits.
  • Attention Is All You Need: the attention mechanism used as the basis for CVA.
